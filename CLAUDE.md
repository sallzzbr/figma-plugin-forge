# CLAUDE.md

This file is an alias entry point for Claude-oriented tooling. The canonical contract and reading hierarchy live in [`AGENTS.md`](AGENTS.md) plus [`docs/`](docs/).

If you are Claude (or any Claude-based assistant), read [`AGENTS.md`](AGENTS.md) first, then follow the reading hierarchy declared there. This file adds nothing Claude-specific on top of `AGENTS.md` — it exists only so tools that look for `CLAUDE.md` by convention can find their way in.

For the adapter matrix and per-assistant notes, see [`docs/guides/assistant-integrations.md`](docs/guides/assistant-integrations.md).
